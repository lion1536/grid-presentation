No Grid utilizamos propriedades row para linhas e column para colunas, e temos uma propriedade para espaçamento:

  • Grid-template-columns
  • Grid-template-rows
  • Row-gap(espaçamento)

Podemos definir tamanhos como por exemplo:

  • Grid-template-columns: 1fr 1fr 1fr;
  • Grid-template-rows: 1fr 1fr;
  • Row-gap: 10px;

Grid-area e Span

O grid-area e Span estão interligados, porque sem um ou outro, não tem como ajustar a largura das colunas, para ajustar a largura, podemos usar uma variedade de medidas, não é algo linear, exemplo:

  • Grid-area: 1 / 1 / span 2 / span 2;

Quando mexemos no segundo valor depois da barra é quando a mágica acontece

Z-index com Overlay

O z-index define a ordem de empilhamento de elementos posicionados (absolute, relative, fixed, ou sticky). Quanto maior o z-index, mais acima o elemento será renderizado na "pilha".

  • Z-index: 10;

Quanto maior o valor do z-index, mais "acima" na pilha de renderização o elemento fica. Isso significa que ele será desenhado por cima de elementos com z-index menor.

  • Grid-auto-rows: 100px;

A propriedade grid-auto-rows define a altura das linhas que o grid cria automaticamente, ou seja, quando não são especificadas manualmente com grid-template-rows.

Grid-template-areas

E é possível mudar o tamanho de cada célula separadamente.

  • Grid-templpate-areas: "header header header" "sidebar content ads" "footer footer footer";

A propriedade grid-template-areas cria uma "máscara" visual do layout, usando nomes para as células.

Auto-fill

  • grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));

auto-fill: Gera automaticamente as colunas. minmax(200px, 1fr): Cada coluna terá no mínimo 200px, mas pode crescer até 1fr (fração do espaço restante).
